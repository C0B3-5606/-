# 🌸 玫瑰花 - 粒子花朵动画系统

一个精美的粒子花朵动画项目，支持多种花朵样式。

## ✨ 特性

- 🌹 玫瑰 - 真实玫瑰形态（花杆、花瓣、花蕊）
- 🪷 莲花 - 浮叶、花瓣、莲蓬
- 🌻 向日葵 - 高茎、多花瓣
- 🌸 百合 - 6瓣长花瓣、雄蕊
- 🌷 郁金香 - 杯形花苞
- 🌺 樱花 - 飘落效果
- 🌼 雏菊 - 螺旋花心

## 🎮 交互功能

- 点击画布：粒子爆发
- 鼠标移动：吸引粒子
- 空格键：暂停/继续
- 支持粒子连线（星座效果）

## 🛠️ 技术栈

- HTML5 Canvas
- JavaScript 原生实现
- Web Audio API（可选音频反应）
